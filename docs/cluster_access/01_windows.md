# Usando Windows

## Descargar terminal de linux (WSL)

Para poder realizar este tutorial en Windows primeramente debemos instalar un sistema Linux en Windows. Este lo podemos encontrar en la *Microsoft store*.

![PIG Platform Logo](../assets/images/logo_pig.png){ style="display: block; margin: 0 auto; width: 500px;" }

En el buscador ingresamos _wsl_ y nos saldran opciones de sistemas linux (distribuciones), se puede eligir la que prefiera el usuario. En este turorial usamos *Ubuntu 20.04.6 TLS*.

![PIG Platform Logo](../assets/images/logo_pig.png){ style="display: block; margin: 0 auto; width: 500px;" }
